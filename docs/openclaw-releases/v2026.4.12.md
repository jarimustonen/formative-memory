# v2026.4.12

**Julkaistu:** 2026-04-13
**Vaikutus:** 🟡 Kohtalainen — plugin loading -muutokset, LM Studio bundled provider, Active Memory -parannuksia

## Pluginiin vaikuttavat muutokset

- **Plugins/loading: manifest-declared activation** (#65120, #65259, #65298, #65429, #65459) — CLI, provider ja channel -aktivointi rajattu manifestin deklaraatioihin; keskitetty manifest-owner-politiikka. **Vaikutus:** Meidän plugin rekisteröi toolit, context enginen ja komennot — kaikki manifestissa deklaroituja. Ei pitäisi vaatia muutoksia, mutta **toimenpide:** varmistettava että meidän `openclaw.plugin.json` manifest kattaa kaikki tarvittavat deklaraatiot uuden tiukemman aktivointipolitiikan alla.

- **Models/providers: LM Studio bundled provider** (#53248) — Bundlattu LM Studio -provider: onboarding, model discovery, stream preload ja **memory-search embeddings**. **Vaikutus:** LM Studio -provider tukee nyt memory-search-embeddingejä. Tämä tarkoittaa uutta embedding-provideria joka näkyy `listMemoryEmbeddingProviders()`-registryssa. Meidän `resolveEmbeddingProvider("auto")` löytää ja käyttää tätä automaattisesti. Hyvä uutinen lokaalien embedding-mallien käyttäjille.

- **Memory/Active Memory: QMD recall default** (#65068) — Active Memory oletuksena käyttää hakua (search) QMD-recallissa, parempaa search-path-telemetriaa. **Vaikutus:** Active Memory kehittyy edelleen. Ei suora vaikutus meidän pluginiin.

- **Gateway: startup/runtime seam split** (#63975) — Startup ja runtime -saumat erotettu elinkaaren ylläpidon helpottamiseksi. **Vaikutus:** Liittyy meidän issue #08 proposal-service-start -ehdotukseen. Gateway startup -rakenteen selkeytyminen voi helpottaa meidän upstream-PR:n hyväksyntää.

- **Gateway/startup: deferred scheduled services** (#65365) — Ajastetut palvelut viivästetään kunnes sidecarit ovat valmiita. Chat-historia gatetetaan sidecar-resumen aikana. **Vaikutus:** Parantaa meidän startup-service-workaroundia — cron-triggerit eivät enää ala ennen kuin sidecarit (mukaan lukien meidän plugin) ovat valmiita.

- **Memory/wiki: Unicode-slugit, tiedostonimirajoitukset** (#64742) — Unicode säilytetään wiki-slugeissa, tiedostonimet rajoitetaan turvallisiin tavupituuksiin. **Vaikutus:** Ei suora vaikutus. Memory-wiki on erillinen.

- **Security: busybox-poisto, tyhjä approver-listan korjaus, shell-wrapper-tunnistus** (#65713, #65714, #65717) — **Vaikutus:** Yleinen turvallisuusparannus. Ei suora vaikutus pluginiin.

- **Dreaming: managed heartbeat -tapahtumat kulutetaan kerran** — Dreaming-narrative-siivous hardennettu, transient session -avaimet eristetty per-workspace. **Vaikutus:** Ei suora vaikutus. Parantaa yleistä heartbeat-luotettavuutta josta meidän cron-triggerit hyötyvät epäsuorasti.
