# v2026.4.14

**Julkaistu:** 2026-04-14
**Vaikutus:** 🟡 Kohtalainen — embedding-normalisointi-korjaus, cron-scheduler-korjauksia, heartbeat/cron-eristys

## Pluginiin vaikuttavat muutokset

- **Memory/embeddings: non-OpenAI provider prefix -säilytys** (#66452) — Embedding-malliviittausten normalisointi ei enää poista non-OpenAI-provider-prefiksejä. **Vaikutus:** Tämä korjaa mahdollisen ongelman jossa Gemini- tai Ollama-embedding-malliviittaukset normalisoituivat väärin. Meidän `resolveEmbeddingProvider()` hyötyy tästä — erityisesti `createGeminiEmbeddingProvider()` saa oikeat mallinimet.

- **Memory/Ollama: restored embedding adapter** (#63429, #66078, #66163) — Sisäänrakennettu `ollama`-embedding-adapteri palautettu memory-coreen, endpoint-aware cache -avaimet. **Vaikutus:** Ollama-embedding-adapteri on nyt taas saatavilla `listMemoryEmbeddingProviders()`-registryssa. Meidän auto-resolution löytää sen. Hyvä uutinen Ollama-käyttäjille.

- **Cron/scheduler: stopped inventing short retries, preserved error-backoff floor** (#66019, #66083, #66113) — Cron-scheduler ei enää keksi lyhyitä uudelleenyrityksiä, virhe-backoff-lattia säilyy. **Vaikutus:** Suora parannus meidän cron-trigger-logiikkaan (issue #21). Cron-ajastetut consolidation- ja maintenance-tehtävät toimivat luotettavammin ilman turhia uudelleenyrityksiä.

- **Gateway/sessions: heartbeat/cron/exec ei ylikirjoita routing-metadataa** (#66073, #63733, #35300) — Heartbeat-, cron- ja exec-käännökset eivät enää ylikirjoita jaetun session routing-metadataa. **Vaikutus:** Kriittinen korjaus meidän cron-triggereiden kannalta. Ennen tätä cron-triggerimme saattoivat korruptoida session-routing-tietoa, mikä johti odottamattomiin sivuvaikutuksiin.

- **Dreaming/memory-core: live queued cron event vaadittu ennen heartbeat hook sweep** (#66139) — Heartbeat-hook-sweep vaatii nyt olemassaolevan cron-tapahtuman. **Vaikutus:** Estää turhia heartbeat-sweeppejä jotka saattoivat triggeroida meidän context enginen tarpeettomasti.

- **Memory/Active Memory: recalled memory hidden untrusted prompt-prefix** (#66144) — Active Memory -muistot siirretty piilotetulle untrusted prompt-prefix -polulle. **Vaikutus:** Ei suora vaikutus. Active Memory kehittyy tietoturvan osalta.

- **Heartbeat/security: force owner downgrade untrusted `hook:wake` events** (#66031) — Ei-luotetut `hook:wake`-tapahtumat pakotetaan owner-downgradeen. **Vaikutus:** Parantaa turvallisuutta heartbeat-triggereiden käsittelyssä. Meidän cron-triggerit saattavat kulkea tämän polun kautta.

- **Telegram/forum topics: human topic names in agent context** (#65973) — Topic-nimet näkyvät nyt agenttikontekstissa ja plugin hook -metadatassa. **Vaikutus:** Meidän plugin voi hyödyntää rikkaampia topic-nimiä kontekstuaalisen muistinhaun parantamiseen (esim. "tämä keskustelu käytiin #projektit-topicissa").

- **OpenAI Codex: `gpt-5.4-pro` forward-compat** (#66453) — **Vaikutus:** Ei suora vaikutus pluginiin.
